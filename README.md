# rimerc

一个自然码双拼用户的Rime配置

## 文件位置

- Windows
  - Weasel: `%APPDATA%\Rime`
- Mac OS X
  - Squirrel: `~/Library/Rime`
- Linux
  - Fcitx5: `~/.local/share/fcitx5/rime/`
  - Fcitx: `~/.config/fcitx/rime`
  - iBus: `~/.config/ibus/rime`
